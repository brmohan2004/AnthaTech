## 🚨 TROUBLESHOOTING

### Not Indexed by Google
1. Check robots.txt for "Disallow: /"
2. Verify noindex tags aren't present
3. Submit sitemap to GSC
4. Request indexing for key pages
5. Check for crawl errors in GSC

### Ranking Drops
1. Check for algorithm updates
2. Analyze competitor changes
3. Review recent site changes
4. Check for technical issues
5. Audit content quality

### Low CTR
1. Rewrite meta titles (more compelling)
2. Improve meta descriptions (add CTAs)
3. Add schema markup for rich snippets
4. Target more specific keywords
5. Improve page titles

### Slow Page Speed
1. Compress images
2. Enable browser caching
3. Minify CSS/JS
4. Use CDN
5. Implement lazy loading
6. Reduce server response time
